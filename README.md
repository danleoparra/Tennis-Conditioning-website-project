# Condicionamento Físico AAACHSA

Website em React + Vite + Tailwind para o guia de condicionamento físico do time de tênis de campo AAACHSA.

## Como rodar localmente

```bash
npm install
npm run dev
```

## Como publicar na Vercel

1. Crie uma conta em https://vercel.com
2. Crie uma conta em https://github.com
3. Crie um novo repositório no GitHub.
4. Faça upload de todos os arquivos deste projeto para o repositório.
5. Na Vercel, clique em "Add New Project".
6. Importe o repositório do GitHub.
7. Framework preset: Vite.
8. Build command: `npm run build`
9. Output directory: `dist`
10. Clique em Deploy.
